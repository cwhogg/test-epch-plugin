Files distilled from these sources:

- `preferred-phrases.md` — extracted from frequency analysis across the source corpus (e.g. "24/7" appears in nearly every Marley external doc; "coordinated, convenient, continuous" / "the future" / "the best way" recur).
- `examples/01-brand-headlines.md` — copied verbatim from HUb deliverable.
- `examples/02-elevator-pitches.md` — copied verbatim from HUb deliverable (Options 1 and 3 retained for each length; Option 2 omitted only to keep file readable, all three are equally canonical).

- No formal "banned-phrases" list either. The banned-phrases file is partly inferred (these phrases don't appear in source corpus) and partly written from the anti-corporate-stock-language principle implicit in the voice.
- No tonal guidance for clinical-protocol comms or for regulatory/legal comms — flagged as a gap.
