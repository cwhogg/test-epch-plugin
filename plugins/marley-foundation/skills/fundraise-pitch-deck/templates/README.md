# Fundraise deck templates

The canonical PPT visual template lives at:
`~/epch-claude/visual brand/Marley_Presentation_MasterTemplate.pptx`

## Workflow

1. Open the master template.
2. Save as `Marley Medical - <Firm Name> - <Date>.pptx`.
3. Build following the canonical arc in `../narrative-arc.md`.
4. Customize per the "Investor-specific customization" guidance.
5. Apply visual brand rules from `../../marley-visual-brand/` — colors, typography, logo placement.
6. After drafting, hand to `chris-critic` per the skill's SKILL.md.

## What this folder does NOT contain

The actual `.pptx` is not duplicated into the plugin — the source-of-truth visual template is the master in `visual brand/`.

If a fundraise-specific layout set (e.g. an "ask slide" template, a "growth-trajectory chart" layout, a "risks paid-down" 2-column layout) ever gets formalized as a reusable slide master, drop it here.
