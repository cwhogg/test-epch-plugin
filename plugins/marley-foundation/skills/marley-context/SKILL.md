# Marley Context — ambient context for Marley-specific positioning

This is a **foundation skill**. It does not produce a deliverable — it provides the company-specific facts and framing that should anchor any Marley artifact.

## When this skill loads

Any task that needs to know who Marley is, what Marley does, what market Marley operates in, what's currently in flight at Marley, or how Marley positions against alternatives. This is broad — most Marley work needs at least some of this context.

## What to do when this skill loads

1. **Read [[company]]** — Marley's mission, what Marley does for a patient, the five-point "How is Marley different" positioning, founding team and history.
2. **Read [[market]]** — TAM logic, ICP, patient demographics, unmet needs, competitive landscape (direct + adjacent), state strategy, payer strategy.
3. **Read [[pipeline]]** when current operational state matters — most recent source data (Nov 2022 investor updates + March 2024 decks). **Note the cutoff date** — anything more recent should be verified before claiming.

## Source-cutoff caveat

The most recent investor update in source is **November 2022**; the most recent deck is **March 2024**. Anything claimed about Marley's state beyond March 2024 is not source-grounded. If the user asks about current state and the answer isn't in source, say so and ask.
