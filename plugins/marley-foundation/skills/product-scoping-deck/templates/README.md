# Product-scoping deck templates

The canonical PPT visual template lives at:
`~/epch-claude/visual brand/Marley_Presentation_MasterTemplate.pptx`

## Workflow

1. Open the master template.
2. Save as `Marley Medical - <Scoping Topic> - <Month Year>.pptx`.
3. Build following the narrative arc in `../narrative-arc.md`:
   - Cover
   - Problem
   - Required attributes
   - Analysis methodology
   - Key learnings
   - Scenario walkthrough (using `../scope-template.md` for each scenario)
   - Cross-scenario summary
   - Appendix
   - END

## What this folder does NOT contain

The actual `.pptx` is not duplicated into the plugin — the source-of-truth visual template is the master in `visual brand/`.

If a scoping-deck-specific layout set (e.g. an "assumptions block + metric output" slide template) ever gets formalized as a reusable slide master, drop it here.
